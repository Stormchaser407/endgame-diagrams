## ðŸ”§ USAGE

### Viewing Diagrams
These `.mmd` files are written in **Mermaid.js** syntax and can be rendered using:
- **Mermaid Live Editor:** https://mermaid.live
- **VS Code:** Mermaid Preview extension
- **Obsidian:** Native Mermaid support
- **GitHub:** Native rendering in markdown files
- **Notion:** Mermaid block support

### Editing Diagrams
1. Open the `.mmd` file in any text editor
2. Modify the Mermaid syntax as needed
3. Increment version number in header comment
4. Test rendering before committing changes
5. Maintain color palette consistency

### Exporting
To export diagrams as images:
1. Open in Mermaid Live Editor
2. Use built-in PNG/SVG export
3. Or use Mermaid CLI: `mmdc -i diagram.mmd -o diagram.png`
